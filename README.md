# Проект на Реакте с авторизацией и регистрацией

Использованы предыдущие наработки по проекту Mesto.

Все запросы на авторизацию, регистрацию и проверку токена работают через сервис `https://auth.nomoreparties.co`. Остальные запросы, не относящиеся к этой проектной работе работают через бэкенд из предыдущих спринтов.
